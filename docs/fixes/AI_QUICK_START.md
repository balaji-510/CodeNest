# 🚀 AI Assistant - Quick Start

## Get Real AI Responses in 3 Steps!

---

### Step 1: Get FREE Groq API Key (30 seconds)

1. Visit: **https://console.groq.com/**
2. Sign up with Google/GitHub
3. Click "API Keys" → "Create API Key"
4. Copy the key (starts with `gsk_...`)

---

### Step 2: Set Environment Variable (10 seconds)

**Windows PowerShell:**
```powershell
$env:GROQ_API_KEY="gsk_paste_your_key_here"
```

**Windows CMD:**
```cmd
set GROQ_API_KEY=gsk_paste_your_key_here
```

**Linux/Mac:**
```bash
export GROQ_API_KEY="gsk_paste_your_key_here"
```

---

### Step 3: Restart Server (20 seconds)

```bash
cd CodeNest/codenest_backend
python manage.py runserver
```

---

## ✅ Done! Test It:

1. Open any problem in CodeNest
2. Click the bot button (🤖) bottom-right
3. Ask: **"Give me a hint"**
4. Get intelligent AI response! 🎉
